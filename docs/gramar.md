# Alna gramar rules 
PROGRAM: STATEMENT*
STATEMENT: ASSIGNMENT | EXPRESSION | VAR_DECLARATION
ASSIGNMENT: [IDENTIFIER | VAR_DECLARATION] '=' EXPRESSION
VAR_DECLARATION: [DATA_TYPE | IDENTIFIER] IDENTIFIER
EXPRESSION: BINARY_EXPRESSION | LITERAL | IDENTIFIER | PARENTHESIZED_EXPRESSION
PARENTHESIZED_EXPRESSION: '(' EXPRESSION ')'
BINARY_EXPRESSION: EXPRESSION OPERATOR EXPRESSION
