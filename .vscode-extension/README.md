# Alna Language Support for VS Code

This extension provides syntax highlighting for the Alna programming language.

## Features

- Syntax highlighting for `.alna` files
- Support for keywords: `if`, `else`, `while`, `for`, `return`, `break`, `continue`
- Type highlighting: `int`, `i8`, `i16`, `i32`, `i64`, `u8`, `u16`, `u32`, `u64`, `float`, `f32`, `f64`, `bool`, `string`, `void`
- Constants: `true`, `false`, `null`
- Operators, numbers, strings, and comments

## Installation

1. Copy the `.vscode-extension` folder to your VS Code extensions directory:
   - Windows: `%USERPROFILE%\.vscode\extensions`
   - macOS/Linux: `~/.vscode/extensions`

2. Restart VS Code

3. Open any `.alna` file to see syntax highlighting

## Development

To work on this extension locally, you can also open the `.vscode-extension` folder directly in VS Code and press F5 to launch a new window with the extension loaded.
